# Checklist UX móvil para trámites online

Checklist práctico para revisar servicios públicos desde móvil.

## Lenguaje claro

- [ ] El objetivo del trámite se entiende en menos de un minuto.
- [ ] Se explica quién puede hacer el trámite.
- [ ] Se indican requisitos antes de empezar.
- [ ] Se indican documentos necesarios.
- [ ] Se evita lenguaje administrativo innecesario.
- [ ] Hay una próxima acción clara.

## Uso desde móvil

- [ ] El texto se lee sin zoom excesivo.
- [ ] Los botones son visibles.
- [ ] El menú es comprensible.
- [ ] Los formularios se pueden completar desde móvil.
- [ ] Los errores se muestran de forma clara.
- [ ] No se obliga a usar ordenador sin aviso previo.

## Formularios

- [ ] Los campos obligatorios están claros.
- [ ] Los mensajes de error explican qué corregir.
- [ ] El usuario puede revisar antes de enviar.
- [ ] Se confirma el envío.
- [ ] Se indica cómo guardar o descargar justificante.

## Resultado esperado

La revisión debe terminar con:
- problemas principales
- impacto en la persona usuaria
- recomendaciones priorizadas
- mejoras rápidas posibles
