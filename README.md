# Auditorías UX de servicios públicos

Auditorías móviles de usabilidad para servicios públicos y trámites online.

Este repositorio reúne revisiones prácticas de páginas web, formularios y servicios digitales públicos desde la perspectiva de una persona usuaria que realiza trámites principalmente desde el móvil.

## Objetivo

Detectar problemas reales de uso y proponer mejoras claras, concretas y priorizadas.

## Estructura de cada auditoría

Cada auditoría debe incluir:

- Servicio o página analizada
- Tarea que intenta realizar la persona usuaria
- Dispositivo y contexto de uso
- Problema observado
- Impacto sobre la persona usuaria
- Evidencia visual o descripción concreta
- Recomendación de mejora
- Prioridad: alta / media / baja
- Versión en lenguaje claro, cuando sea útil

## Criterios de análisis

- Claridad del lenguaje
- Accesibilidad móvil
- Facilidad para encontrar información
- Errores o bloqueos del proceso
- Formularios difíciles de completar
- Falta de instrucciones
- Exceso de pasos
- Diseño no adaptado al móvil
- Dificultades para personas migrantes o con baja competencia digital

## Auditorías previstas

- Servicios de empleo
- Trámites de formación
- Páginas de inscripción a cursos
- Citas administrativas
- Formularios públicos
- Servicios municipales
